什么是 Flask？

Flask 是 Python 社区最流行的轻量级 Web 框架之一。与 Django 这样的"大而全"框架不同，Flask 只提供核心功能——路由、模板渲染和请求/响应处理，其余部分由开发者自由选择扩展。

这种设计哲学让 Flask 非常适合以下场景：

  • 快速原型开发
  • 小型到中型 Web 应用
  • RESTful API 服务
  • 微服务架构

快速开始

安装 Flask 只需要一行命令：

pip install flask

创建一个 app.py，写入以下代码：

from flask import Flask

app = Flask(__name__)

@app.route("/")
def hello():
    return "Hello, World!"

运行 python app.py，打开浏览器访问 http://localhost:5000，你就能看到 "Hello, World!"。

核心概念

Flask 的核心概念非常少，掌握它们就能上手开发：

路由（Routing） — 通过 @app.route() 装饰器将 URL 绑定到 Python 函数。

模板（Templates） — 使用 Jinja2 模板引擎生成动态 HTML 页面。

请求与响应 — Flask 自动解析 HTTP 请求，并支持多种响应格式（HTML、JSON、文件等）。

小结

Flask 的核心理念是"简洁而不简单"。从一行代码打印 "Hello, World!" 到支撑数百万用户的应用，Flask 都能胜任。关键在于，它把选择权交给了你。
